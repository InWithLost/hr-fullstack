Список всех страниц фронтенда и их описание:

1. / (index.vue)
   Главная страница сайта. Обычно содержит основную информацию о сервисе, преимущества, краткое описание возможностей и ссылки на ключевые разделы.

2. /login (login.vue)
   Страница входа пользователя. Форма для авторизации по логину/почте и паролю.

3. /register (register.vue)
   Страница регистрации нового пользователя. Форма для создания аккаунта.

4. /contacts (contacts/index.vue)
   Страница с контактной информацией компании, формой обратной связи или картой.

5. /blog (blog/index.vue)
   Список всех статей блога, новости, публикации.

6. /blog/[slug] ([slug].vue)
   Динамическая страница отдельной статьи блога. [slug] — уникальный идентификатор публикации.

7. /jobs (jobs/index.vue)
   Список всех вакансий, фильтрация и поиск по вакансиям.

8. /jobs/[slug] ([slug].vue)
   Динамическая страница отдельной вакансии. [slug] — уникальный идентификатор вакансии.

9. /legal/offer (offer.vue)
   Страница с публичной офертой.

10. /legal/personal-data (personal-data.vue)
    Страница с политикой обработки персональных данных.

11. /legal/privacy (privacy.vue)
    Страница с политикой конфиденциальности.

12. /legal/terms (terms.vue)
    Пользовательское соглашение (правила пользования сервисом).

13. /profile (profile/index.vue)
    Личный кабинет пользователя, основная информация и настройки.

14. /admin (admin/index.vue)
    Главная страница административной панели.

15. /admin/blog (admin/blog.vue)
    Управление статьями блога (создание, редактирование, удаление).

16. /admin/jobs (admin/jobs.vue)
    Управление вакансиями (создание, редактирование, удаление).

17. /admin/users (admin/users/index.vue)
    Управление пользователями платформы.

18. /specialties (specialties/index.vue)
    Список всех медицинских специальностей.

19. /specialties/[slug] ([slug].vue)
    Динамическая страница отдельной специальности.

Описание построено на основании структуры папки pages. Для уточнения назначения каждой страницы можно посмотреть содержимое соответствующего файла.
